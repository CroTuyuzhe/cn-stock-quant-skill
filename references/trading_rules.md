# 量化交易规则模板

## 标准入场规则（全部满足）

1. PE(TTM) < 5年中位数（估值偏低）
2. RSI(14) < 35（超卖确认）
3. 收盘价 < MA60（破位中期均线）
4. 20日跌幅 > 5%（超跌确认）

## 标准离场规则（满足任一）

1. RSI(14) > 70（超买）
2. 20日涨幅 > 10%（获利了结）
3. 收盘价 > MA20 × 1.05（偏离均线过多）

## 止损规则

- 跌破60日最低价 → 无条件止损
- PE跌破历史10%分位且基本面未变 → 加仓而非止损

## 止盈规则

- RSI(14) > 70 → 减半仓
- 20日涨幅 > 10% → 减至底仓
- 股息率 < 3% → 清仓

## 回测方法

1. 遍历历史数据，标记买入/卖出信号
2. 买入后持有至卖出信号触发
3. 统计：胜率、平均收益、盈亏比、最大单笔亏损、平均持仓天数
4. 基准：同期买入持有收益

## 阈值调整规则

- 不同股票波动特征不同，RSI/动量阈值需按历史回测优化
- 最佳阈值 = 历史回测中夏普比率最高的阈值
- 扫描范围：RSI超卖 20~40, 动量阈值 -3%~-10%
