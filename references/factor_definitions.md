# 因子定义与计算方法

## 因子总览

| 类别 | 因子名 | 信号方向 | 数据需求 |
|------|--------|---------|---------|
| 反转 | rsi_reversal | RSI<30超卖→看多 | close |
| 反转 | bollinger_deviation | 低于下轨→看多 | close |
| 反转 | mean_reversion | 短期MA偏离长期MA→回归 | close |
| 动量 | macd_momentum | MACD柱为正→看多 | close, high, low |
| 动量 | price_momentum | N日涨幅/波动率→看多 | close |
| 动量 | trend_strength | 短期MA>长期MA→上升趋势 | close |
| 量能 | volume_momentum | 放量上涨→强看多 | close, volume |
| 量能 | obv_trend | OBV斜率为正→资金流入 | close, volume |
| 波动 | volatility_regime | 短期波动<长期波动→低风险 | close |

## 因子计算

### 反转因子

**RSI反转因子：** `(50 - RSI(14)) / 50`，归一化到[-1,1]。正值=超卖看多，负值=超买看空。

**布林偏离因子：** `(中轨 - 收盘) / (上轨 - 下轨)`，裁剪到[-2,2]。正值=价格低于中轨（超卖）。

**均值回归因子：** `(MA20 - MA5) / MA20`，短期偏离长期越大，回归概率越高。

### 动量因子

**MACD动量因子：** `MACD柱状图 / ATR(14)`，用ATR归一化。正值=上升动量。

**价格动量因子：** `N日收益率 / N日波动率`，波动率归一化后的动量。裁剪到[-3,3]。

**趋势强度因子：** `(MA5 - MA20) / MA20 * 50`，裁剪到合理范围。

### 量能因子

**量价配合因子：** `量比 * sign(价格变化)`，量价同向为正。

**OBV趋势因子：** `(OBV - OBV均线) / OBV标准差`，资金流向趋势。

### 波动率因子

**波动率状态因子：** `1 - 短期波动率/长期波动率`，正值=低波动（低风险）。

## ICIR回测

每个因子计算IC（信息系数）= 因子值与未来N日收益的相关系数。
ICIR = IC均值 / IC标准差。|ICIR| > 0.3 视为有效因子。
